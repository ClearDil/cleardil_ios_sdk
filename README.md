## Installation

cleardil_kyc_pod is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'cleardil_kyc_pod', :git => 'https://github.com/ClearDil/kyc_pod.git', :tag => '1.0.0'

```
